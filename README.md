# Enzo Almeida Academic Website

Quarto source files for `https://enzoadealmeida.github.io`.

## Local Preview

Install Quarto, then run:

```bash
quarto preview
```

## Render

```bash
quarto render
```

The rendered site is written to `docs/`, which can be used for GitHub Pages.

## GitHub Pages

Create a public GitHub repository named `enzoadealmeida.github.io`, push this folder, and configure GitHub Pages to serve from the `docs/` folder on the main branch.
